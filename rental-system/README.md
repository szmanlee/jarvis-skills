# 🏢 石岩房租管理系统

专业的全栈房租收租统计管理系统，为石岩地区19户出租屋量身打造。

## ✨ 系统特色

- 🏠 **19户精准管理** - 101-310房间全覆盖
- 💰 **智能费用计算** - 电费1元/度，水费7元/吨自动计算
- 📊 **专业财务报表** - 月度汇总、统计分析、Excel导出
- 🌐 **现代化界面** - 响应式设计，支持移动端访问
- 💾 **SQLite数据库** - 轻量级，零配置部署
- 🔄 **实时数据更新** - 修改即保存，数据实时同步

## 🚀 快速开始

### 1. 环境要求
- Node.js 14+
- npm 6+

### 2. 安装依赖
```bash
cd rental-system
npm install
```

### 3. 初始化数据库
```bash
npm run init-db
```

### 4. 启动系统
```bash
npm start
```

### 5. 访问系统
打开浏览器访问: http://localhost:3000

## 📊 数据结构

### 📈 财用项目
| 项目 | 单价 | 计算方式 |
|------|------|----------|
| 房租 | 按房间 | 固定金额 |
| 电费 | 1元/度 | (当月-上月)×单价 |
| 水费 | 7元/吨 | (当月-上月)×单价 |
| 管理费 | 20-40元 | 按房型 |
| 燃气费 | 0元 | 按实际用量 |

### 🏘️ 房间分布
- **101-104**: 一楼 (4户)
- **201-204**: 二楼 (4户)  
- **301-310**: 三楼 (10户)
- **总计**: 19户

## 🎯 API接口

### 月份管理
- `GET /api/months` - 获取月份列表
- `POST /api/months` - 创建新月份
- `GET /api/months/:id` - 获取月份详情

### 费用管理
- `PUT /api/charges/:id` - 更新费用记录
- `GET /api/tenants` - 获取住户列表
- `GET /api/settings` - 获取系统设置
