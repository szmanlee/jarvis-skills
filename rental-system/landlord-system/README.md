# 房东专用本地收租管理系统

## 🏠 系统概述

这是一个专为个人房东设计的本地化收租管理系统，无需服务器，双击即可使用。系统包含19户固定结构，支持完整的水电费管理和租客信息管理。

## 📁 文件结构

```
landlord-system/
├── index.html          # 登录页面
├── management.html     # 管理页面主界面
└── README.md          # 使用说明文档
```

## ⚡ 系统功能

### 🔐 登录验证
- 独立的登录页面，未登录无法访问管理系统
- 登录状态自动过期（2小时）
- 密码可在代码中修改

### 🏠 房间管理
- 按楼层清晰展示19个房间
- 支持编辑租客信息（姓名、身份证、电话）
- 租金和押金管理
- 合同到期日期管理

### 📸 水电表管理
- 上传水电手抄表照片
- 模拟OCR自动识别房间号、电表读数、水表读数
- 手动输入水电表读数
- 自动计算用量和费用
- 电费：1.2元/度（可修改）
- 水费：3.5元/吨（可修改）

### 💰 缴费管理
- 已缴/未缴状态标记
- 实时统计已缴费房间数
- 一键切换缴费状态

### ⏰ 提醒功能
- 合同到期前30天自动提醒
- 到期房间红色感叹号标识

### 📊 统计功能
- 总房间数自动统计
- 已租房间数统计
- 月租总收入计算
- 已缴费数量统计

## 🚀 快速开始

### 方法1：直接打开（推荐）
1. 双击 `index.html` 文件
2. 输入用户名：admin，密码：admin123
3. 登录成功后自动跳转到管理页面

### 方法2：本地服务器（更稳定）
```bash
# 如果有Python：

# 如果有Node.js：
npx serve .

# 然后访问：http://localhost:8080
```

## 🔧 自定义设置

### 修改登录密码
在 `index.html` 中找到并修改：
```javascript
const DEFAULT_USERNAME = 'admin';        // 修改用户名
const DEFAULT_PASSWORD = 'admin123';     // 修改密码
```

### 修改水电价格
在 `management.html` 中找到并修改：
```javascript
electricRate: 1.2,  // 电费单价 元/度
waterRate: 3.5,   // 水费单价 元/吨
```

### 修改登录过期时间
在 `management.html` 中找到并修改：
```javascript
const twoHours = 2 * 60 * 60 * 1000;  // 修改过期时间（毫秒）
```
