# Data Agent Learnings

## Chart Selection Guide

| Data Type | Comparison | Distribution | Relationship | Composition |
|-----------|------------|--------------|--------------|-------------|
| **Categories** | Bar chart | Histogram | Scatter | Pie chart |
| **Time Series** | Line chart | Area chart | - | Stacked area |
| **Multi-variable** | Grouped bar | Box plot | Bubble chart | Sankey |

## Statistical Basics

- **Mean:** Average (sensitive to outliers)
- **Median:** Middle value (robust)
- **Mode:** Most frequent value
- **Std Dev:** Spread of data
- **Correlation:** Relationship strength (-1 to 1)

## Data Cleaning Steps

1. Check for missing values
2. Identify outliers
3. Standardize formats (dates, currencies)
4. Remove duplicates
5. Validate data types

## Visualization Best Practices

1. Start with a clear question
2. Keep it simple
3. Use color purposefully
4. Label axes and legends
5. Add context (benchmarks, goals)
6. Source your data
