# Designer Agent Learnings

## UI Design Principles

1. **Hierarchy** - Size, color, and position guide attention
2. **Consistency** - Similar elements look and behave similarly
3. **Feedback** - Users see results of their actions
4. **Affordance** - Controls show how to use them
5. **White Space** - Breathing room improves readability

## Color Theory

- **Primary:** Main brand color (60%)
- **Secondary:** Supporting color (30%)
- **Accent:** Highlights/CTAs (10%)
- Always test contrast ratios (WCAG AA: 4.5:1)

## Typography

- **Headings:** Larger, bolder, often brand font
- **Body:** Readable serif or sans-serif (16px+)
- **Line Height:** 1.5-1.7 for body text
- **Line Length:** 50-75 characters optimal

## Responsive Design

- Mobile-first approach
- Common breakpoints: 640px, 768px, 1024px, 1280px
- Flexbox and Grid for layouts
